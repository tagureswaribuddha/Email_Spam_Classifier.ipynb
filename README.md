# Email Spam Classifier using Machine Learning

## Project Description
This project is a beginner-friendly machine learning project that classifies SMS messages as spam or not spam. It uses TF-IDF Vectorization to convert text messages into numerical features and a Naive Bayes classifier to make predictions.

## Technologies Used
- Python
- Google Colab
- Pandas
- scikit-learn
- TF-IDF Vectorizer
- Naive Bayes Classifier

## Dataset
The project uses the SMS Spam Collection Dataset.

## Steps Followed
1. Loaded the SMS spam dataset.
2. Checked the dataset structure and label distribution.
3. Converted text labels into numerical labels.
4. Split the data into training and testing sets.
5. Used TF-IDF Vectorizer to convert text into numerical features.
6. Trained a Naive Bayes classification model.
7. Evaluated the model using accuracy, classification report, and confusion matrix.
8. Tested the model using custom messages.

## Result
The model achieved approximately 97% accuracy on the test dataset.
